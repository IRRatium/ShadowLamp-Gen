


# 💡 ShadowLamp-Gen (v2.0 PRO)

**ShadowLamp-Gen** — это профессиональный генератор 3D-моделей теневых светильников. Загрузите любую 2D-картинку (логотип, текст, фото), и алгоритм на основе SDF (Signed Distance Fields) сгенерирует готовую к печати цилиндрическую STL-модель.

Алгоритм физически-корректно рассчитывает прохождение света сквозь толщу пластика так, чтобы на столе сформировалась идеальная ровная неискаженная тень.

## ✨ Ключевые возможности

- 🚀 **Математическая SDF Геометрия**: Модель строится не из кубиков-вокселей, а с помощью непрерывных полей. Края отверстий абсолютно гладкие (Anti-aliased).
- 🌉 **Умные Опорные Столбы**: Автоматически прошивают рисунок, не позволяя "островам" (например, центрам букв "О") отвалиться при 3D-печати. Размер тени от столбов остается неизменным на полу независимо от высоты светильника!
- 🧴 **Геометрия Горлышка**: Настраиваемая база с переходом в конус (трапецию) и верхнее кольцо для удобного крепления патрона лампочки.
- 🎨 **Рендер в реальном времени**: Встроенный 3D-движок (Three.js) позволяет покрутить модель в браузере и увидеть, как двигается физически-корректная тень при изменении высоты лампочки.

## 🛠 Установка и запуск

Проект написан на **Python (FastAPI)** и не требует сложных зависимостей вроде PyTorch.

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/IRRatium/ShadowLamp-Gen.git
   cd ShadowLamp-Gen
   ```

2. Установите зависимости (рекомендуется использовать виртуальное окружение `venv`):
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите локальный сервер:
   ```bash
   python app.py
   ```

4. Откройте браузер по адресу: **http://127.0.0.1:8000**

## 🖨 Советы по 3D-печати

- **Цвет пластика**: Используйте **Чёрный (непрозрачный)** PLA или PETG. Светлые и полупрозрачные пластики будут пропускать свет через стенки, и контрастная тень на столе не получится.
- **Поддержки (Supports)**: Печатайте лампу вертикально. Большинство отверстий распечатаются мостами (bridging) без поддержек, но для сложных узоров включите древовидные поддержки (Tree supports) только с внешней стороны.
- **Толщина стенок**: В веб-интерфейсе по умолчанию стоит 2.0 мм. Для 3D-принтеров с соплом 0.4 мм это ровно 5 периметров.

## ⚙️ Стек технологий
